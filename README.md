## git分支提交规范

    在使用git提交时，请建立名字为自己ID的分支。
    
    并在完成之后将自己分支的更新推送到dev分支上。
    
    请不要修改master分支。
    
    自己的文件放在/[你的ID] 目录中

## 有关纪律/活动安排




>* 大概每周会要求一次报告




>* 有困难或者问题及时提出，小组成员间要多进行交流（尤其是当两个人任务相关性大时）




>* 定时查看QQ群以及WonderList（奇妙清单），以便通知以及对进度的掌握
